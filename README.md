# Autonomous Data Analyst

##  Project Overview

The **Autonomous Data Analyst** project focuses on automating the end-to-end data analysis workflow. This project takes raw data as input, performs data cleaning, exploratory data analysis (EDA), and generates meaningful insights with minimal manual intervention. The goal is to simulate how a data analyst approaches a dataset and extracts insights using Python.

This project demonstrates practical data analysis skills that are relevant for internships and entry-level Data Analyst roles.

---

##  Objectives

* Automate basic data cleaning and preprocessing
* Perform Exploratory Data Analysis (EDA)
* Identify trends, patterns, and insights from data
* Reduce manual effort in repetitive analysis tasks

---

## Tools & Technologies Used

* **Python**
* **Pandas** – data manipulation
* **NumPy** – numerical operations
* **Matplotlib / Seaborn** – data visualization
* **Jupyter Notebook** – analysis and experimentation

## ▶️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/rakshitha45k/Autonomous-Data-Analyst.git
   ```
2. Install required libraries:

   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
4. Run `analysis.ipynb`
